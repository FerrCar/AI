Aiuto Codice

"Aiuto Codice", supporto tecnico e formativo

Guida rapida: esempi di snippet in HTML, CSS, JavaScript o Python

Risorse utili: link a documentazione, tutorial, editor online

FAQ tecniche: risposte a errori comuni (es. come faccio il PHP?, Le query?)

Esempi pratici:

Variabili e operazioni:

<?php
$nome = "Mario";
$eta = 18;
echo "Ciao, $nome. Hai $eta anni.";
?>

Condizioni e cicli:

<?php
if ($eta >= 18) {
    echo "Sei maggiorenne.";
} else {
    echo "Sei minorenne.";
}

for ($i = 0; $i < 5; $i++) {
    echo "Numero: $i <br>";
}
?>

Connessione al database MySQL (MySQLi):

<?php
$conn = new mysqli("localhost", "user", "password", "nome_database");
if ($conn->connect_error) {
    die("Connessione fallita: " . $conn->connect_error);
}
echo "Connessione riuscita!";
$conn->close();
?>

Consiglio: usa sempre pratiche sicure.
